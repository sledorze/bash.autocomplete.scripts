
BASH.AUTOCOMPLETE
=================
HaXe autocomplete scripts for bash.


INSTALL
-------

    # Copy scripts 
    cp [SCRIPT] /etc/bash_completion.d

    # Source
    source /etc/bash_completion.d/[SCRIPT]

    # Test
    [PROGRAM] -[TAB]
